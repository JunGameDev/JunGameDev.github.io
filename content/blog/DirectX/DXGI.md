# <b> DXGI 란? </b>
- DirectX Graphics Intrastructure
- DirectX의 구성 요소중 하나
- DX10에서 처음 도입됨
<br>
# <b> DXGI가 하는 일 </b>
- DirectX graphics runtime과 무관한 로우 레벨 작업들을 관리한다.
- 그래픽스 API와 그래픽스 커널 사이의 맵핑을 제공한다.
- 아래와 같은 테스크들을 제어하기 위한 오브젝트들을 제공한다.
  - 그래픽 어뎁터 및 모니터 열거
  - 표시 모드 열거
  - 버퍼 형식 선택
  - 프로세스 간 자원 공유
  - 렌더링 된 프레임을 화면에 출력
